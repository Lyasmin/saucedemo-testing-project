# QA Portfolio - SauceDemo E-commerce Testing

This project aims to demonstrate my Quality Assurance skills through a complete testing cycle of the SauceDemo website.

## Project Structure

* [**01 - Requirements**](./01-Requirements/1%20-%20Requirements%20-%20SauceDemo.pdf): Functional and non-functional requirements documentation.
* **02 - Test Plan:** Strategic planning for the testing cycle.
* **03 - Test Scenarios:** High-level testing scenarios.
* **04 - Test Cases:** Detailed test cases with steps and expected results.
* **05 - Test Execution:** Reports of executed tests and evidence.
* **06 - Bug Report:** Documentation of found issues.

## Tools Used
* Manual Testing
* GitHub (Documentation & Version Control)
* Google Docs / PDF (Reporting)
